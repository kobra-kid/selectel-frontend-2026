# Selectel Test

Тестовое задание на стажировку в Selectel на позицию фронтенд-разработчика. Мини-приложение на Angular 19.

## Стек

- Angular 19
- TypeScript
- HTML, CSS

## Функциональность

- **Главная страница** — приветствие и переход на страницу меню
- **Страница меню** — список разделов в сайдбаре, при выборе раздела отображаются его позиции с чекбоксами и ценами
- **Заголовок** — показывает название активного раздела, количество выбранных позиций и их суммарную стоимость

## Запуск

```bash
npm install
ng serve
```

Приложение будет доступно по адресу `http://localhost:4200`
